What is Bootstrap?

Bootstrap is a free, open source toolkit and is the most popular CSS framework developed by twitter to create mobile-first responsive website. It contains HTML, CSS, and JavaScript-based template for grid layout, navigation, buttons, forms, and other functions.

Easy to start

The developer just needs to understand the basic knowledge of HTML and CSS will be able to create a mutual and elegant website by using Bootstrap.

Responsive design

Bootstrap helps us create responsive applications automatically. This also means the webpage will automatically adapts to different screen sizes that is desktop, laptop, and mobile, etc.
